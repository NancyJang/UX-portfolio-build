TAIPEI PUBLIC LIBRARY WEBPAC REDESIGN


Improving Online Library Experience Through Contextual Search

The BRIEF


台北市公共圖書館網站每月瀏覽約100萬次，使用量約10萬,是分享信息與提供社區民眾圖書查閱書籍服務的主要工具。「館藏查詢」該網站附屬於「臺北市立圖書館網站」下的一個單元，並非獨立的SG。網站更新頻率不頻繁,維護方式為館員蒐集，人工建置。網站主要內容為：查閱、預約書籍，線上電子資源，查詢個人借閱相關資料等等。

PROBLEM & GOAL


在圖書館擔任志工期間，我發現圖書館用戶對網站的看法與網站本身存在重大困惑，因公共圖書館性質，資訊眾多，分頁零散,電子書、電子期刊等分頁連結是另闢單元與其並列，在這個項目中需要解決的問題之一。另外，這次主要為改善PC端網頁設計,我與其中一位館員進行工作，首先了解用戶使用上的經驗,先針對現有的問題進行研究分析，並向圖書館和網站維護專員提出更改的建議。

UX設計對於公共圖書館仍然比較新，為了促進館員參與，我對用戶體驗設計主題進行了介紹。在整個過程中，我的目標是針對用戶主要需求，我通過重新設計網站地圖、優化網頁設計，讓用戶能清楚查閱資訊，並使流程流暢和多功能簡化。

  • To track the usage of the library main website.
  • To track visitors' behaviors.
  • To make suggestions for improving user experiences.
  • To establish the most effective way for redesigning the website.

USER REASEARCH


為了促進圖書館館員參與和組織對使用者為中心的設計實踐的廣泛了解，我對用戶體驗設計主題進行訪談線上問卷。 這些執行結果有助於提高館員對於網站易用性的了解，並讓圖書館員工在維護網站方面所起的各種作用。 同時，我也將改善的設計流程整合到圖書館的現有管理工作流程中，並向圖書館員工及館長說明。

HOW TO DO?

USER INTERVIEWS


我們從八個圖書館網站用戶獲得了關於他們在使用圖書館網站上的觀點的反饋。在進行訪談與測試後，進行分析，確定了網站可用性研究的關鍵。

PERSONA

另外我們請用戶進行一些網站使用的測試。用戶主要給出了五個一致的反饋。

USABILITY PROBLEMS

1.non-obvious signs: 我感到困惑，我想查閱我的借書紀錄，從哪裡登入？
- 根據網站用戶行為分析，四位用戶中就有三位有這個困擾，網站顏色主頁過於混亂，引發了用戶找不到登入的欄位 ，即使有文字說明，但依然不顯能顯而易見。＊採取行動的按鈕區域應該要能吸引使用者注意，避免和其他功能互搶風采而造成的視覺混亂

2.Search function is difficult to use:“查閱功能不好使用”
- 大多數用戶沒有使用高級搜索的選項，直接搜索書名或關鍵字，但搜尋結果包含許多關聯性較低的書籍。 即使書籍名稱被指定為搜索關鍵字，結果仍然包含許多關聯性較低的書籍。

3.Inconsistent logo and domain name:“我覺得他們看起來不像同一個網站”
-至少有四種不同顏色的logos和兩種網站的名稱，不同的視覺體現及資訊架構會讓用戶產生困惑。＊網站logo需掌握統一的原理，在多樣性設計下提煉為一個主要表現體

4.Unable to figure out how to start CVS service: “超商借書如何操作？” - 超商借書圖示佔了網頁版面的5/1，但點擊後並沒有說明如何操作，用戶無法了解如何這個功能。 此功能對於許多用戶來說是吸引人的宣傳行動按鈕。有四分之一用戶在點擊後試圖在網頁內找尋流程說明，但因無法得到信息而放棄，降低了網站用戶的信心。

5.The main navigation was ambiguous：“我覺得迷惘，我無法在網站上瀏覽我需要的資訊” - 網站主選單的導航是模糊的，讓用戶在尋找目標功能時，需要過濾很多不相干的資訊。另外不是所有的連結和選單按鈕都是一致的，“電子資源查詢”和”新書通報服務“兩個功能項目點擊後都是另開啟的獨立頁面，放在選單列並不適合。
