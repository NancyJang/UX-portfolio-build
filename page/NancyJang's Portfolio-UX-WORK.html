Klokers手錶銷售之 App 規劃及設計

使用者研究與win9 APP設計

The BRIEF


klokers手錶台灣代理商想在實體通路銷售加入平板輔助，期望能將銷售的過程記錄下來，改善並創新通路服務體驗。由於前期開發預算有限，想先針對實體通路買錶用戶與銷售人員進行調查，找出痛點設計ＡＰＰ平板介面。

GOAL


在現今科技發達的時代，錶的市場競爭越來越激烈，專案中以用戶經驗、使用姓、服務為導線，從一線銷售人員的經驗值上著力，帶入全新體驗的銷售經營策略。從研究到介面設計打造一個全新的服務流程，突破過去錶店的傳統銷售模式，提供錶店人員以及顧客更完善的購買體驗與互動模式。

調查主要內容為：各實體通路目標用戶，現有宣傳管道，合作方式可能性，銷售方法。

USER REASEARCH

User journeys and Invovled parties journeys assumption

＊洞察：klokers為新的錶款,商品在實體通路還是為一個核心的階段，仍需要開發新的服務方法來吸引更多的客戶群。

＊前期準備：通路銷售狀況訪談提綱和可能性之服務觸點：
為釐清Klokers手錶的銷售通路及針對手錶銷售的服務需求，前往目前銷售通路誠品、三創生活園區、頑石鐘錶、進行訪查。針對顧客其脈絡背景進行實際觀察和紀錄，包括銷售員在服務上的困惑和問題。

USER BACKGROUND ANALYSIS

＊主要痛點： 買方想詢問更多手錶細節、銷售員不清楚客人所問的問題、銷售員不在場、下訂流程過於繁瑣、品牌商欲了解客人反應與問題

＊初步構想：
初步發想涉及用戶樣本，為了幫助一開始的原型製作。我們舉辦了一個簡短的研討會，讓關係人員通過我們的初始方案來釐清平板功能到使用介面。

根據搜集的問題進行分析，選影響手錶銷售服務的關鍵因素，並整合符合使用者情境與需求擬定可執行的步驟，並利用卡片分類法找出介面發想的幾個重要的功能，包含幾個大方向：

  • Menu

    1、活動檔期手錶組合、錶帶介紹輔助

  • Print

    2、銷售與詢問紀錄的列印

  • Order

    3、填寫訂單

  • Share

    4、以IG攝影圖像誘導客戶分享信息

Service blueprint


Results


在前期調查與初步執行下，我們替klokers手錶未來開發完成了前期評估與幾個目標：

  • 服務創新

    1、加入新穎數位的銷售方式，不同過去在各實體通路的服務行為

  • 企業獲利

    2、提高市場成長率，增加話題

  • 經營創新

    3、透過科技給顧客專業的體驗
